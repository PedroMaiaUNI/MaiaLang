# MaiaLang
Exercício de Aspectos Teóricos da Computação. Construção de "compilador" utilizando Jflex e JCup.

Para executar esse projeto:

1. Instale jflex-full-1.9.1.jar
2. Instale java-cup-11b.jar
3. Coloque os dois arquivos em uma pasta intitulado "lib" na raiz do projeto
4. Rode o arquivo run.bat (Autoria de Prof. Ronaldo)
